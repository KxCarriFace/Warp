
WARP

A lightweight CLI tool that lets you create aliases for directories and jump to them instantly.

Warp helps you navigate your filesystem faster by replacing long paths with simple commands.

Example:

warp add proj ~/Documents/projects/my-project
warp to proj

instead of typing long paths every time, you can warp directly to them.


Features:
- Create aliases for directories
- Store aliases in a JSON configuration
- validate paths before saving
- Prevents duplicate aliases or paths
- store metadata like:
    - creating timestamp
    - usage count
    - tags
    - descriptions
